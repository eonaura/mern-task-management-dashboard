# MERN Task Management Dashboard

## 📌 Project Overview
A full-stack Task Management Dashboard built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application allows users to manage their daily tasks efficiently with a clean and responsive interface.

## ✨ Features
- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Complete/Pending
- Search Tasks
- Filter Tasks
- Dashboard Statistics
- Responsive User Interface

## 🛠️ Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- Mongoose
- Axios
- CSS

## 🚀 How to Run

### Backend
bash
cd backend
npm install
npm run dev


### Frontend
bash
cd frontend
npm install
npm run dev
